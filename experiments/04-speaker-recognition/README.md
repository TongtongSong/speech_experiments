# 实验04
## 实验内容
本次实验利用公开数据库实现声纹识别系统并进行验证。该实验旨在贯通课程学习中声纹识别的相关内容。通过实现前端特征提取、深度神经网络、池化层、损失函数等必要步骤，掌握声纹识别的整体流程，并了解部分优化方法。
